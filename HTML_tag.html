和文档章节相关的标签

在head当中的标签是用户一般不会看到的标签；
从body 标签开始，就是用户会看到的内容。

header, nav, aside, article, section, footer, hx都是html5新增的标签。主要用来表示页面中的一些章节结构

文档的章节从大块儿上分，通常是三部分：header, body 和footer。
body部分中又包括了sidebar, 和主体内容部分。 ps:在html5之前，通常都是用div来分隔这些区域。在html5中用了更加语义化的标签来表示：

  • <header>: 标题，logo，搜索表单。也可以表示一些独立章节的头部。h1-h6。重要性的差别。
  • <nav>: 导航。任何有导航性质的内容。可以有多个导航。
  • <article>: 主体内容，独立的可重复的结构：评论，文章等。可以嵌套，表示相关（博客评论嵌套在博客文章当中） <section>: 内容的专题组，一般带一个标题，相邻的section一般有相关性。
  • <aside>: 侧边栏，广告，工具性内容。
  • <footer>:表示章节尾部：版权信息，相关链接等。也可以用于独立内容的尾部。

和文本相关的标签

  • 超链接 <a></a>：作用：
    创建指向另一个文档的链接：technique movie finance
    <a href = "http://tech.163.com/">technique</a> <a href = "http://ent.163.com/movie/" target = "_self">movie</a>
    <a href ="http://money.163.com" target ="_blank">finance</a>
    跳转方式可以有target属性控制，若target = _self, 则在当前的页面显示，若为target = _blank, 则在新建的页面显示。若不写target,则默认在当前页面显示。target 也可以是一个名称，则页面会在目标的名称中打开：
    < a href ="./site/js.html" target ="inner" > </a> <iframe frame border="0" name="inner" ></iframe>
    创建文档内部的锚点:
    <a href="#pay">buy</a> <div id="pay">buy</div> 注意链接值要和id值一直，才能跳转到相应内容（跳转后，地址栏也会加上#值。也可以用这个功能完成『回到顶部』；
    连接到Email地址：
    <a href="mailto:shy@sina.com">contact</a> 类似功能是在手机上点击手机号码时自动跳出拨号软件，代码如下： <a href="tel:111111">111111</a> ps:邮箱地址可以多个，用逗号分隔。
  • 强调一些内容：
    <strong>:重要性上的强调，比em更强烈，粗体 <em> ：语义上的强调，斜体。
    有些时候我们并不是想强调某个内容，而是想让它有不一样的样式。一般就会把它放在<span>标签中。span标签本身没有任何语义，但可以结合class赋予它样式。
  • 换行
    一行文字<br>第二行
  • 引用：
    cite引用出处（会变成斜体）；q引用某一段文字，说的话。（会加上引号）code代码；b摘要中的关键字，产品名称等（会变成粗体）；i斜体表示的内容，技术术语等。
