网易微专业---课程笔记

前端入门 Web front-end Introduction

这篇文章主要介绍了前端是什么， 前端页面的发展历史，如何开发一个复杂的系统，以及前端的工作范围。 This is the introduction of what is web front-end, history, how to develop complex system, and responsibilities.

什么是web系统 what is web system

web系统是部署在web服务器上的为客户端提供服务的系统。这个系统要足够复杂，包含了服务器和客户端两部分。客户端通常是指浏览器。 web system is always deployed on the web server and provides service for client-end. This system is always very complex, and contains server side and client side. Client-end in web systems usually refers to web browsers.

Note:

  • Servers and clients have data exchanged. 服务器端和客户端之间可以进行数据交互。
  • Data refers to contents in the browser. 客户端的数据是跑在浏览器当中的内容。
  • Web front-end/UI is the tool to reveal these data through web browser. 前端和UI就是为了在浏览器中呈现出来这些数据。

web系统的发展史 The history of web system

Web 1.0

Web 1.0时代以内容网页为主，例如公司官网或者产品的介绍页。 Web1.0 depends on the "website's contents", such as company's official website or introduction web-page of a certain product or service.

在1998年，AJAX技术出现，2005年大批基于AJAX技术的应用出现，非常知名的包括Google Maps。随着AJAX异步请求技术的兴起，网页可以通过scripts在不刷新页面的情况下向服务器请求并更新内容。因此，迎来了Web 2.0的时代。 AJAX was created in 1998 and has been widely used since 2005. At that time, there were lots of internet applications made use of AJAX, the most famous application is Google Maps. With the popularity of AJAX asynchronous request, web pages can be updated without refreshing the pages. Therefore, there comes a new era--web 2.0

Web 2.0

Web 2.0时代产品趋向应用化。之后HTML 5出现，新的标准在十年之内被广泛熟知。 Web2.0 intends to build internet into applications. After that, HTML 5 was created and became popular within 10 years.

Web 3.0

Web 3.0 以D3图表库和网页版虚拟现实为代表。 Web 3.0 contains two famous representative that are D3 and RT virtual reality.

页面制作 web page visual design

包括HTML, CSS, PS。用到配色，图片，图表，距离及高度等。 Including HTML, CSS and PS, and make use of color, picture, chart, width and length.
