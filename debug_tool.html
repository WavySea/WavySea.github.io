开发页面的工具：

  • 文本编辑器：Sublime, Notepad ++, EditPlus

  • IDE集成开发环境: 集成了js调试工具。

推荐sublime Text的理由:
跨平台，启动快；
多行选择，方便操作；
使用插件;
snippets(省去了一些快捷键的操作)；
支持VIM兼容模式；

常用快捷键：
1. 查找 Ctrl +P + : 行号 ——定位到行；(Ctrl + G)
2. Ctrl +P + @ 符号 ——定位到具体符号（选择器等）；(Ctrl + R)
3. Ctrl +P + # 关键字 ——定位到具体关键字；(Ctrl + ;)
4. 命令面板 Ctrl + Shift + P 模糊匹配；Ctrl D/Ctrl+shift+L（多行选中）;
5. Alt + F3选中全部变量；

常用插件： (先要安装Package Control;)
Emmet
DocBlockr;
SideBarEnhancements Terminal

常用插件的快捷使用方式：

  • Ctrl N新建文件----- ！+Tab 页面框架就搭建好了---- div#nav>ul>li?4>a 按Tab

  • 通过TOOL—-new snippet新建代码片段，添加代码片段关键字：<tabTrigger>hello</tabTrigger>,添加代码片段使用的文件范围： <scope>text.html</scope> —-保存 在想引入的文件里输入关键字+回车就直接引入代码片段；

  • 在preference---user setting中，把ignored_packages中的内容删掉，若没有这一段代码，则添加一个空的代码段："ignored_packages":[ ]

开发页面的工具

以chrom 为例：
F12

  • Elements: HTML + CSS; —-曾删查改
  • Network: 可以看到网站的网络请求，与后端调试接口时使用
  • Sources:可以看到网站所有的资源（css,图片，js等）可以设置断点来调试js
  • Console: 可以直接写js代码调试，也会输出一些内容，与source面板结合
  • resources：网站的本地资源（cookie, storage等）

  • timeline, profiles, audits: 调试页面性能；

  • 放大镜—-选中想查看的元素—可以直接从Elements面板修改

  • 加号—新加规则

IE：
IE8以上版本 自带调试工具
IE6：安装IETester查看效果 ； 如果要调试，需要安装原生IE6浏览器和IEdevelopToolBar;
仿真功能---可以选择查看调试的版本
